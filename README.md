## 🏆 CAS Case Competition – 1st Prize Winner

### Risk-Based Pricing Framework for Student Renter’s Insurance

A data-driven actuarial pricing project that develops an individual-level pricing framework for student renter’s insurance.  
The project integrates exploratory data analysis, claim frequency and severity modeling, and Monte Carlo simulation to estimate expected loss and support risk-based underwriting decisions.

📄 **Project Report:**  
[CAS Case Competition Report](./CAS%20-%20Team%2024.pdf)

💻 **Project Code:**

- `CAS Case Competition（EDA）.py` – exploratory data analysis  
- `frequency.py` – claim frequency modeling  
- `severity (5).Rmd` – claim severity modeling  
- `monte.ipynb` – Monte Carlo simulation for loss estimation
